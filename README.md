# AI Voice Assistant

A React-based voice assistant with WebGL orb visualization powered by Google Gemini AI.

## Quick Start

```bash
npm install
cp .env.example .env
# Edit .env with your Google Apps Script endpoint
npm run dev
```

## Features

- Real AI responses via Google Gemini 2.5 Flash
- Voice recognition and text-to-speech
- WebGL orb animation with visual feedback
- Multi-key system for reliability

## Browser Support

Chrome, Edge, Safari (requires Web Speech API support)
