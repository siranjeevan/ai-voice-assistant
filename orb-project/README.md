# Orb Background Project

This is a standalone React project containing the Orb background component extracted from `react-bits`.

## Getting Started

1.  Navigate to the project directory:
    ```bash
    cd orb-project
    ```

2.  Install dependencies:
    ```bash
    npm install
    ```

3.  Run the development server:
    ```bash
    npm run dev
    ```

## Files

-   `src/components/Orb/Orb.jsx`: The main Orb component using `ogl`.
-   `src/components/Orb/Orb.css`: Styles for the component.
